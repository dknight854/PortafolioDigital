Opinion Sobre Las Clases

Diseño Web

Esta Clase me ha gustado mucho ya que aprendi cosas como disñar paginas, desde como poner imagenes de fondo, agregar estilos a imagenes, cuadros, botones y todo eso see puede lograr desde css ya que para eso fue echo, para darle estilo a la pagina html. He aprendido que se puede hacer una infinidad de cosas en nuestra web, todo usando nuestra imaginacion y creatividad. Admiro a mi compañero Elias y Alejandro, ya que a ellos le asignaron el desafio y honor de poder crear la pagina web de mi colegio San Vicente de Paul.

Programación

Esta clase tambien me ha gustado, aunque se me ha hecho muy dificil, eso se debe a que no preste la suficiente atencion para poder comprender pero es una clase muy buena en la cual se aprenden un sin fin de cosas. Me gusta la programacion por el hecho de que si tenemos el conocimiento podemos hacer cosas como programar sistemas para bancos, cooperativas o incluso sistemas de seguridad, todo es cuestion de poner atencion y tener interes en la clase.
